Raspberry Pi

What does it do?

Raspberry Pi is a very small (credit card sized) single board computer that connects to a conventional mouse, keyboard and monitor. It was created by an educational charity based out of the UK by the same name, Raspberry Pi foundation. Since its launch in 2012 there have been many iterations and upgrades and as of today, different products and accessories but Raspberry Pi foundation has always worked with the same vision. Make computing, programming and coding cost effective, accessible, and achievable for all people regardless of age and experience (Raspberry Pi Foundation-About us).

Other than the Raspberry Pi 4 model B (its newest model) Some of the other products and accessories that the charity offer is, Raspberry Pi Pico (a new micro controller board), Raspberry Pi 400 (The Raspberry Pi built into a keyboard and of course, the previous models. The Raspberry Pi functions as a computer or laptop, however it is basic. Besides computing, the Raspberry pi can interact with the outside world meaning it has almost endless applications and is limited only to the imagination and this is what really makes the board stand out. Some reported projects are, a retro arcade game, Robotics, doorbell/intercom system, broadcast a FM radio station, a web server, twitter bot, language translator the list is endless. A recent project where the Raspberry Pi 4 was used to refurbish a 1940’s Philco radio shows how this new technology can be used to rejuvenate old technology. Check out the report 1940s Philco radio sings again with new Raspberry Pi heart - Raspberry Pi ( Reported by Ashley Whittaker 2023). With all the new accessories and parts that are launching future applications are exciting to say the least.
